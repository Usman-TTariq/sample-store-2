# sample-store-2

Sample Store 2 — a Next.js coupon and deals site with Supabase backend.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Supabase

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Copy `.env.example` to `.env.local` and add your Supabase credentials.
